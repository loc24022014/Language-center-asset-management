# Nền tảng quản lý kho/tài sản có phân quyền và báo cáo cho trung tâm ngoại ngữ

## Giới thiệu

Hệ thống hỗ trợ quản lý tài sản và kho vật tư tại trung tâm ngoại ngữ, giúp theo dõi số lượng tài sản, phân quyền người dùng và thống kê báo cáo.

## Công nghệ sử dụng

* Java Spring Boot
* SQL Server
* HTML/CSS/Bootstrap
* GitHub
* Docker

## Chức năng chính

* Đăng nhập
* Phân quyền người dùng
* Quản lý tài sản
* Quản lý kho
* Báo cáo thống kê
* Nhật ký hoạt động
